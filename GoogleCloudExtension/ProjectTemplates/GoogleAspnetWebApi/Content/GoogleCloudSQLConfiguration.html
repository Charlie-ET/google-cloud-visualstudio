Google Cloud SQL

Google Cloud SQL is a fully-managed database service that makes it easy to set-up, maintain, manage and administer your relational MySQL databases in the cloud. Hosted on Google Cloud Platform, Cloud SQL provides a database infrastructure for applications running anywhere.

This ASP.NET template uses the Google Cloud SQL relational MySQL database.

To configure this application to use Cloud SQL, edit the Web.config file found at the root of this application and update the database connection string to reflect the username, password, IP, and database of your Cloud SQL database.

Web.config 
<connectionStrings>
    <!--
      Google Cloud SQL database configuration

      To configure this application to use Cloud SQL, change the connectionString attribute and
      replace the placeholders to reflect the username, password, IP, and database of your Cloud SQL database.

      For example, suppose your IPv4 address is 173.194.230.44, your database is dotnetdb, your username
      is dotnetuser, and your password is secret123. Then the connectionString attribute would look like this:

        connectionString="Server=[IP Address];Database=dotnetdb;Uid=dotnetuser;Pwd=secret123"
    -->
    <add name="DefaultConnection"
     connectionString="Server=[IPv4 IP Address];Database=[Database name];Uid=[User name];Pwd=[User password]"
     providerName="MySql.Data.MySqlClient" />
</connectionStrings>
